A 3D interactive experience where a Raccoon collects hearts to reveal memories, culminating in a meeting with a special Pig.

## Controls
- **WASD / Arrow Keys**: Move the Raccoon
- **Mouse**: Rotate Camera
- **Walk into Hearts**: Collect them to reveal a memory
- **Walk to Pig**: Start the final dialogue (after collecting all hearts)

## Development
This project uses:
- React + Vite
- React Three Fiber (3D)
- Zustand (State Management)
- TailwindCSS (Styling)
